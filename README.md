> GeoLocator is a Python tool to find geolocation information from IP addresses quickly and accurately. Perfect for learning, small projects, or network troubleshooting.

---

## Features

- Find geolocation details of an IP  
- Display city, region, country, and ISP information  
- Fast and lightweight  
- Easy-to-use command-line interface

# Install dependencies
